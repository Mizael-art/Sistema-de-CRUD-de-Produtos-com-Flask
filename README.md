# 💻 Sistema de CRUD de Produtos com Flask

Este projeto é um sistema web simples de gerenciamento de produtos, desenvolvido com Python e Flask como parte do meu aprendizado em programação.

---

## 🚀 Funcionalidades

- 📦 Cadastro de produtos
- 📋 Listagem de produtos
- ✏️ Edição de produtos
- 💾 Armazenamento de dados em JSON

---

## 🛠 Tecnologias utilizadas

- Python
- Flask
- HTML
- JSON

---

## 🧠 O que aprendi

- Como criar um backend com Flask
- Como criar rotas e páginas web
- Como integrar frontend (HTML) com backend
- Como manipular arquivos JSON
- Como estruturar um CRUD completo
